금주의 도전과제
===========

화차 내부에 각자 폴더를 만든 후, 자신의 폴더에 자신있는 언어로 작성할 것.

`회차 > 아이디 > 제목.코드`


코드 상단에 제출 코드 주소 넣을 것.

```swift
// 제출 코드
// https://www.acmicpc.net/source/24685767
```

질문은 `Issues` 에 올릴 것.
